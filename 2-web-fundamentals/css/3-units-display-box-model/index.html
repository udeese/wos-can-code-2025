The Display Property

  • inline
    • They like neighbors (if there's room, inline elements share the same line
    • Width and height have no effect
  • block
    • They don't like neighbors (they take up the full width of the parent container)
    • width and height have an effect
  • inline-block
    • A combination of both
    • Leaves a sliver of whitespace if there is whitespace in your HTML
